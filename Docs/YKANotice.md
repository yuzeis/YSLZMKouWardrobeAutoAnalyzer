# 使用边界与隐私

本工具是非官方、只读、Windows 优先的本地报告工具。

- 只被动观察本机游戏客户端已产生的网络流量和进程信息。
- 不发包、不重放、不注入、不抽卡，也不控制微信或微信小程序。
- 原始抓包、运行日志、报告和微信导出数据只保存在本机，程序没有上传功能。
- 最终报告和对应微信导出数据落盘后，程序会自动清理该会话的原始抓包文件。
- 会话证据可能含本机路径、进程信息和网络端点。公开分享前请自行检查并脱敏。
- 微信导入码由报告推导，导入动作必须由用户在小程序中手动完成。
- 紧凑文本和二维码只在本机生成；本程序不修改或操作微信小程序。
- 小程序导入会覆盖当前账号记录。使用导入码前应先在小程序中导出现有数据。

本工具不会绕过账号认证、付费、抽卡概率、游戏限制或小程序权限。

## 第三方组件

Windows 单文件发行版包含 Scapy、psutil、qrcode 和 Pillow；Npcap 是单独安装的系统级外部依赖。组件与许可证信息见根目录 `THIRD_PARTY_NOTICES.md`，各组件仍适用其自身许可证。
